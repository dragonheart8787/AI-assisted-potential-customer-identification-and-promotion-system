## 🚀 如何使用

### 快速開始
1. **安裝 Node.js** (如果尚未安裝)
   - 下載地址：https://nodejs.org/
   - 選擇 LTS 版本

2. **運行安裝腳本**
   ```bash
   # 雙擊運行
   install-app.bat
   ```

3. **啟動應用程式**
   ```bash
   # 雙擊運行
   start-app.bat
   ```

### 啟動選項
- **選項 1**：開發模式 (包含開發者工具)
- **選項 2**：生產模式 (標準運行)
- **選項 3**：打包應用程式 (創建安裝檔)

### 應用程式選單
- **檔案選單**：開啟不同頁面、匯出/匯入數據
- **工具選單**：系統設定、清除快取、重新整理
- **幫助選單**：使用指南、關於

### 快捷鍵
| 功能 | 快捷鍵 |
|------|--------|
| 開啟主系統 | Ctrl+1 |
| 開啟 AI 聊天 | Ctrl+2 |
| 開啟帳號管理 | Ctrl+3 |
| 開啟系統測試 | Ctrl+4 |
| 匯出資料 | Ctrl+E |
| 匯入資料 | Ctrl+I |
| 系統設定 | Ctrl+, |
| 重新整理 | Ctrl+R |
| 使用指南 | F1 |
| 退出 | Ctrl+Q |
