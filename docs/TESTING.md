## 🚀 快速開始

### 1. 啟動測試伺服器
```bash
python -m http.server 8000
```

### 2. 開啟測試頁面
在瀏覽器中訪問：
- **測試啟動頁面**: `http://localhost:8000/start-test.html`
- **主系統**: `http://localhost:8000/index.html`
- **測試報告**: `http://localhost:8000/test-report.html`

## 🎯 測試方法

### 自動測試
1. 開啟主系統頁面
2. 系統會自動執行全面測試
3. 查看瀏覽器控制台的測試結果

### 手動測試
1. 在主系統頁面找到「🧪 系統測試」區域
2. 點擊相應的測試按鈕：
   - 🚀 執行全面測試
   - 🤖 測試AI功能
   - 📤 測試真實傳送
   - 🤖 測試智能機器人

### 測試報告
1. 訪問測試報告頁面
2. 查看詳細的測試結果和建議
3. 根據建議進行系統優化

## 🔧 故障排除

### 常見問題

#### 1. 伺服器無法啟動
```bash
# 檢查端口是否被佔用
netstat -an | findstr :8000

# 使用不同端口
python -m http.server 8001
```

#### 2. 測試失敗
- 確保所有JavaScript文件正確載入
- 檢查瀏覽器控制台是否有錯誤訊息
- 重新整理頁面後再次測試

#### 3. AI功能無法使用
- 檢查瀏覽器是否支援現代JavaScript功能
- 確保網路連接正常
- 清除瀏覽器快取後重試

### 測試結果解讀

#### 成功率 90%+
- 🎉 系統運行正常
- 可以開始使用所有功能

#### 成功率 70-90%
- ⚠️ 部分功能可能需要配置
- 檢查失敗的測試項目
- 根據建議進行調整

#### 成功率 < 70%
- ❌ 系統需要修復
- 檢查錯誤日誌
