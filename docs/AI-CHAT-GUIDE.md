## 🔧 故障排除

### 常見問題

#### 1. 頁面無法載入
- 檢查伺服器是否啟動
- 確認端口8000未被佔用
- 重新啟動批處理文件

#### 2. AI功能無回應
- 重新整理頁面
- 檢查瀏覽器控制台錯誤
- 確認所有JavaScript文件載入

#### 3. 測試失敗
- 查看測試結果詳情
- 檢查系統狀態指示器
- 重新執行測試

### 解決方案

#### 伺服器問題
```bash
# 檢查端口佔用
netstat -an | findstr :8000

# 終止佔用進程
taskkill /f /im python.exe

# 重新啟動
python -m http.server 8000
```

#### 瀏覽器問題
- 清除瀏覽器快取
- 使用無痕模式測試
- 嘗試不同瀏覽器
