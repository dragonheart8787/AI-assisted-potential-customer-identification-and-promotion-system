## 🔄 資料流程圖

### 1. 用戶登入流程
```
用戶啟動系統
    ↓
載入主介面 (index.html)
    ↓
選擇登入平台 (Twitter/LinkedIn/Instagram/Facebook)
    ↓
選擇登入方式 (手動/Google OAuth)
    ↓
┌─────────────────┐    ┌─────────────────┐
│   手動登入      │    │   Google OAuth   │
│   輸入帳密      │    │   選擇帳號      │
│   驗證登入      │    │   授權應用      │
└─────────────────┘    └─────────────────┘
    ↓                        ↓
    └────────┬───────────────┘
             ↓
    儲存認證資訊 (localStorage)
             ↓
    更新UI狀態 (登入成功)
             ↓
    啟用推銷功能
```

### 2. AI推銷流程
```
啟動推銷任務
    ↓
載入目標客戶列表
    ↓
AI分析客戶資料
    ↓
┌─────────────────────────────────────┐
│         AI內容生成引擎               │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ │
│  │投資人模板│ │CEO模板  │ │合作夥伴模板│ │
│  └─────────┘ └─────────┘ └─────────┘ │
└─────────────────────────────────────┘
    ↓
個人化內容生成
    ↓
A/B測試版本創建
    ↓
┌─────────────────────────────────────┐
│        批量發送引擎                 │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ │
│  │延遲控制 │ │錯誤重試 │ │速率限制 │ │
│  └─────────┘ └─────────┘ └─────────┘ │
└─────────────────────────────────────┘
    ↓
發送到各平台
    ↓
記錄發送結果
    ↓
更新CRM資料
    ↓
觸發自動跟進
```
