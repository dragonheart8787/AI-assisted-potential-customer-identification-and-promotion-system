# AI 推銷助手桌面應用程式使用指南

## 概述

AI 推銷助手現在已經轉換為一個真正的桌面應用程式，使用 Electron 框架構建。這意味著您可以像使用任何其他桌面軟體一樣使用它，無需瀏覽器或網路連接。

## 安裝步驟

### 1. 前置需求

- **Node.js** (版本 18.x 或更高)
  - 下載地址：https://nodejs.org/
  - 選擇 LTS (長期支援) 版本

### 2. 安裝應用程式

#### 方法一：使用安裝腳本（推薦）
```bash
# 雙擊運行
install-app.bat
```

#### 方法二：手動安裝
```bash
# 安裝依賴
npm install

# 啟動應用程式
npm start
```

### 3. 打包成安裝檔
```bash
# 打包應用程式
npm run build
```

打包完成後，安裝檔將位於 `dist` 資料夾中。

## 故障排除

### 常見問題

#### 1. 應用程式無法啟動
**解決方案：**
- 檢查 Node.js 是否正確安裝
- 運行 `npm install` 重新安裝依賴
- 檢查防火牆設定

#### 2. 依賴安裝失敗
**解決方案：**
- 檢查網路連接
- 清除 npm 快取：`npm cache clean --force`
- 使用管理員權限運行

#### 3. 打包失敗
**解決方案：**
- 確保所有依賴已正確安裝
- 檢查 `package.json` 配置
- 查看錯誤日誌

#### 4. 數據無法匯出/匯入
**解決方案：**
- 檢查檔案權限
- 確保目標路徑可寫入
- 檢查檔案格式是否正確

### 錯誤代碼

| 錯誤代碼 | 說明 | 解決方案 |
|----------|------|----------|
| ENOENT | 檔案不存在 | 檢查檔案路徑 |
| EACCES | 權限不足 | 使用管理員權限 |
| ENOTFOUND | 模組未找到 | 重新安裝依賴 |
| ECONNREFUSED | 連接被拒絕 | 檢查網路設定 |
