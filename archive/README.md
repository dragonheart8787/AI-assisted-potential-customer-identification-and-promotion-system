# archive／封存區

此目錄存放**非主線展示**之檔案，避免與正式學習歷程入口混淆。

- **舊版首頁**：`index-fixed.html`、`index-real-google.html` 等。
- **除錯／過渡頁**：`debug-*.html`、`google-login-debug.html`、`quick-test.html` 等。
- **已併入推廣中心之舊模組頁**：如 `real-time-analytics-dashboard.html`（KPI 請使用根目錄 `kpi-dashboard.html`）。
- **Electron 選單仍可能指向的封存頁**：`ai-chat-test.html`、`secure-account-manager.html`（僅供需要時開啟）。

正式展示請一律使用：**`app-new.html`**（`index.html` 會轉址）。
