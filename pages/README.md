# pages／正式子頁

此目錄為 **推廣中心（`app-new.html`）的子頁與工具頁**，請由主介面連結進入，**不要當成獨立產品入口**。

- `crm-interface.html` — CRM／Pipeline
- `kpi-dashboard.html` — 唯一正式 KPI 儀表板
- `ai-settings.html`、`api-settings.html` — 設定
- `ai-promotion-workflow.html` — 推廣工作流程
- `ai-knowledge-base.html` — 知識庫
- `scoring-explained.html` — 評分說明
- `system-overview.html` — 架構說明
- `test-app-new.html` — 瀏覽器端能力測試

腳本路徑為 `../` 指向根目錄之 `.js` 模組。
