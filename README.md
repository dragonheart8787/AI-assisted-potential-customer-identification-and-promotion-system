# 科技高層自動訊息發送系統

一個專為與科技產業高層主管建立聯繫而設計的自動化訊息發送系統，整合了對話式推銷策略、智能對話功能、真實帳號發送能力和完整的AI對話模型，並配備高級AI強化模組。

## 🚀 系統特色

### 核心功能
- **自動化訊息發送**：支援多平台訊息發送（X、LinkedIn、Instagram、Facebook、微博）
- **真實帳號整合**：支援使用真實社群媒體帳號進行實際發送
- **對話式推銷策略**：整合WIIFM原則、黃金一分鐘法則和心理觸發點
- **智能對話增強**：提供三種對話模式（專業、輕鬆、策略）
- **完整AI對話模型**：智能建議、內容生成、情感分析、優化功能
- **高級AI強化模組**：深度情感分析、智能優化、對話流程、個性匹配、情境感知
- **個人化內容**：根據每位領袖的特色自動調整訊息內容
- **平台適配**：根據不同社群平台特性調整訊息格式
- **排程發送**：支援延遲發送和批量發送功能
- **發送記錄**：完整的訊息發送歷史和狀態追蹤

### AI對話模型功能
- **🤖 AI智能建議**：基於領袖特色和對話模式提供內容建議
- **✨ AI內容生成**：自動生成個人化訊息內容（問候語、價值主張、互動內容、結束語）
- **🔍 AI智能分析**：分析訊息的情感、關鍵詞、意圖、相關性和優化建議
- **⚡ AI內容優化**：自動優化訊息內容，提升效果
- **🗣️ AI對話助手**：即時AI對話功能，提供智能建議和幫助

### 高級AI強化模組功能
- **🎭 深度情感分析**：多維度情感分析，包括情感基調、意圖檢測、語氣分析
- **⚡ 智能內容優化**：基於ML模型的內容優化，提升可讀性和影響力
- **🔄 對話流程規劃**：自動生成完整的對話階段和時間線
- **👤 個性匹配分析**：分析領袖個性並提供個人化溝通策略
- **🧠 情境感知分析**：綜合分析時機、平台、產業、個人和外部情境

### 對話增強功能
- **智能建議**：基於領袖特色和對話模式提供內容建議
- **情境分析**：分析發送時機、語氣、相關性和風險
- **回應生成**：自動生成適合的回應內容
- **對話模式**：專業模式、輕鬆模式、策略模式三種選擇

### 真實帳號功能
- **多平台支援**：Twitter、LinkedIn、Instagram、Facebook、微博
- **API整合**：完整的API端點和認證流程
- **批量發送**：同時向多個平台發送訊息
- **連接測試**：測試帳號連接狀態
- **安全存儲**：本地安全存儲API金鑰

## 🎯 對話式推銷策略

### 核心原則
1. **WIIFM (What's In It For Me)**：明確說明對對方的價值
2. **黃金一分鐘**：在開頭60秒內抓住注意力
3. **心理觸發點**：運用社會認同、稀缺感、權威等原則

### 對話模式
- **專業模式**：適合正式商業溝通，使用結構化語言
- **輕鬆模式**：適合建立個人關係，使用自然對話
- **策略模式**：適合長期關係建立，使用深度分析

### AI對話模型特色
- **智能建議**：基於領袖特色、對話模式、當前內容提供建議
- **內容生成**：自動生成問候語、價值主張、互動內容、結束語
- **情感分析**：分析訊息的情感基調（積極、消極、中性）
- **關鍵詞識別**：自動提取重要關鍵詞
- **意圖檢測**：識別訊息的主要意圖（介紹、提案、詢問、邀請）
- **相關性分析**：分析內容與目標領袖的相關程度
- **優化建議**：提供具體的內容優化建議

### 高級AI強化模組特色
- **深度情感分析**：多維度分析情感、意圖、語氣、緊急程度、正式程度
- **智能內容優化**：基於ML模型的內容優化，提升可讀性和影響力
- **對話流程規劃**：自動生成完整的對話階段和時間線
- **個性匹配分析**：分析領袖個性並提供個人化溝通策略
- **情境感知分析**：綜合分析時機、平台、產業、個人和外部情境

### 策略功能
- **自動個人化**：根據領袖特色調整內容
- **平台適配**：根據社群平台調整格式
- **策略提示**：即時提供策略建議
- **預覽功能**：發送前預覽個人化內容

## ⚠️ 重要提醒

### 真實帳號使用
- 請確保擁有相應平台的開發者帳號和API權限
- 遵守各平台的API使用限制和服務條款
- 妥善保管API金鑰和存取權杖
- 建議在測試環境中先進行測試

### AI對話模型使用
- AI功能基於規則引擎和模式識別
- 建議內容僅供參考，請根據實際情況調整
- 情感分析結果可能不完全準確
- 定期更新AI模型以提升準確性

### 高級AI強化模組使用
- 高級AI功能基於ML模型和深度分析
- 個性匹配分析僅供參考，實際溝通需靈活調整
- 情境感知分析結果可能不完全準確
- 建議結合實際情況使用AI強化功能

### 合規使用
- 本系統僅供學習和策略規劃使用
- 請遵守各平台的服務條款和隱私政策
- 確保訊息內容真實、合法且不涉及垃圾訊息
- 尊重對方的隱私權和回覆意願

### 安全注意事項
- API金鑰僅存儲在本地，不會上傳到伺服器
- 建議定期更換API金鑰
- 不要在公共環境中輸入敏感資訊
