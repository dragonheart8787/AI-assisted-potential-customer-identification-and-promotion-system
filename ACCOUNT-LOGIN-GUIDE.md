# 🔐 帳號登入系統使用指南

## 📋 系統概述

這個系統現在已經整合了完整的帳號登入功能，讓您可以登入自己的社群媒體帳號並使用真實訊息發送功能。

## 🚀 主要功能

### 1. **多平台帳號登入**
- **Twitter/X**：支援用戶名/Email + 密碼登入
- **LinkedIn**：支援Email + 密碼登入
- **Instagram**：支援用戶名 + 密碼登入
- **Facebook**：支援Email + 密碼登入

### 2. **帳號狀態管理**
- 即時顯示登入狀態（未登入/已登入/連接中）
- 自動保存登入資訊到本地存儲
- 支援多帳號同時登入

### 3. **連接測試功能**
- 測試所有已登入帳號的連接狀態
- 模擬真實API連接測試
- 顯示連接結果通知

### 4. **安全功能**
- 本地密碼存儲（不會上傳到伺服器）
- 登出功能清除所有敏感資訊
- 一鍵清除所有登入資料

## 🛠️ 使用步驟

### 第一步：啟動系統
1. 雙擊 `start-account-system.bat` 啟動腳本
2. 系統會在瀏覽器中自動打開 `index.html`

### 第二步：設定用戶資料
1. 在「用戶資料設定」區域填寫您的個人資訊
2. 點擊「💾 保存資料」按鈕

### 第三步：登入社群媒體帳號
1. 在「社群媒體帳號登入」區域選擇要登入的平台
2. 輸入您的用戶名/Email和密碼
3. 點擊對應的登入按鈕
4. 等待登入完成（會顯示「連接中...」狀態）

### 第四步：測試連接
1. 登入完成後，點擊「🧪 測試所有連接」按鈕
2. 系統會測試所有已登入帳號的連接狀態
3. 查看測試結果通知

### 第五步：使用系統功能
1. 選擇目標科技領袖
2. 選擇對話模式和訊息模板
3. 啟用對話式推銷策略
4. 使用AI功能生成和優化內容
5. 發送訊息到已登入的平台

## 🔧 功能說明

### 登入狀態指示器
- **未登入**：紅色背景，表示帳號未登入
- **連接中**：黃色背景，表示正在登入中
- **已登入**：綠色背景，表示登入成功

### 帳號管理
- **登入**：輸入憑證並登入帳號
- **登出**：清除特定平台的登入資訊
- **測試連接**：驗證帳號連接狀態
- **清除所有**：一次性清除所有登入資料

### 通知系統
- **成功通知**：綠色背景，顯示操作成功
- **錯誤通知**：紅色背景，顯示操作失敗
- **警告通知**：黃色背景，顯示注意事項
- **資訊通知**：藍色背景，顯示一般資訊

## ⚠️ 重要提醒

### 安全注意事項
1. **密碼安全**：密碼僅存儲在本地，不會上傳到任何伺服器
2. **帳號保護**：請勿在公共環境中輸入敏感資訊
3. **定期登出**：使用完畢後建議登出帳號
4. **清除資料**：離開前可使用「清除所有登入」功能

### 使用限制
1. **模擬登入**：目前系統使用模擬登入功能進行測試
2. **API整合**：真實API整合需要相應平台的開發者帳號
3. **使用條款**：請遵守各平台的服務條款和API使用限制

### 故障排除
1. **登入失敗**：檢查用戶名和密碼是否正確
2. **連接異常**：使用「測試所有連接」功能檢查狀態
3. **頁面錯誤**：重新整理頁面或重新啟動系統
4. **資料丟失**：檢查瀏覽器是否支援localStorage
